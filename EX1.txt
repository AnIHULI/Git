Посмотреть где я:
	pwd

Создать папку dir_X:
	mkdir dir_X

Перейти в созданную папку dir_X:
	cd dir_X

Перейти в родительскую директорию:
	сd ..

Создать 3 папки:
	mkdir dir_1_1 dir_2_2 dir_3_3

Перейти в любую из созданных папок:
	cd dir_2_2

Создать 5 файлов (3 txt, 2 json):
	touch 1.txt 2.txt 3.txt 1.json 2.json

Создать 3 папки:
	mkdir dir_x1 dir_x2 dir_x3

Вывести список файлов, директорий в текущей директории:
	ls -la

Открыть любой txt файл:
	cat > 1.txt

Открыть любой txt файл и записать что-нибудь в него:
	cat > 1.txt
	Test Text
	Ctrl + C

Перейти в директорию выше:
	cd ..

Переместить файлы в директорию:
	mv 1.txt 2.txt 3.txt dir_x3/

Копировать файлы в директорию:
	cp 1.json 2.json dir_x1/

Найти файл по имени:
	find dir_x2/11.txt

Вывести строку из файла:
	grep 'Barsik' 7.json
	
	Output:
		"name" : "Barsik",

Вывести несколько первых строк (5) из текстового файла:
	head -5 7.json

Вывести несколько последних строк (5) из текстового файла:
	tail -5 7.json

Посмотреть содержимое длинного файла:
	less 7.txt
	/person_100

Вывести дату и время:
	date +'%d-%m-%y %T'

Отправить запрос через терминал:
	curl https://www.google.com
